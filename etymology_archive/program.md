# program
## program (n.)
1630年代，“公告”，源自於拉丁語 programma “公告，法令”，來自希臘語 programma “書面公告”，源自 prographein “公開寫作”，由 pro “向前”（見 [pro-](https://www.etymonline.com/tw/word/pro-)）和 graphein “寫作”（見 [-graphy](https://www.etymonline.com/tw/word/-graphy)）組成。  

“音樂會上演奏曲目的書面或印刷清單”這一含義最早見於1805年，保留了最初的意義。 “廣播節目”這一含義始於1923年。  

“明確的計劃或方案，預先準備或宣佈的操作方法或程序線路”的一般意義最早見於1837年。 “一系列編碼指令，指導計算機執行特定任務”的計算機含義始於1945年。  

“音樂所暗示的對象或事件”的含義始於1854年（program music 最早見於1877年）。拼寫爲 programme，在現代使用中源自法語，始於19世紀早期，最初特別是指“戲劇節目單”的意義。  

---
## program (v.)
1889年，“編寫程序說明”（現已過時）； 1896年作爲“按照程序安排”，源自 [program](https://www.etymonline.com/tw/word/program)（名詞）。  

對於計算機，“使其自動按照預定方式調節”始於1945年； 這一概念在1963年被擴展到動物身上，比喻爲“訓練其按照預定方式行爲”； 在1966年被用於人類。相關詞彙： Programmed； programming。
