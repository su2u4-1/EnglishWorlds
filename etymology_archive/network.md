# network
## network (n.)
1550年代，“線，電線等的網狀排列，以網或網狀物的外觀形成的任何東西”，來自 [net](https://www.etymonline.com/tw/word/net)（n.）+ [work](https://www.etymonline.com/tw/word/work)（n.）。 “任何複雜的，交織的系統”的擴展意義始於1839年（最初是指通過河流，運河和鐵路進行運輸）。 “多個發射器的廣播系統”的意義來自1914年； “相互連接的人羣”的意義是1934年心理學術語。  

---
## network (v.)
1887年，"用網覆蓋"，源自 [network](https://www.etymonline.com/tw/word/network)（名詞）。從1940年開始用於"在（廣播）網絡上播出"； 1972年用於計算機； 到1982年用於人，意爲"與他人互動以交換信息和建立聯繫"。相關詞彙： Networked； networking。
