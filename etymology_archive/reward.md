# reward
## reward (n.)
14世紀中期，“應得的報酬，應得的懲罰”，來自盎格魯-法語和古北法語 reward, rouwart，是從 rewarder（見 [reward](https://www.etymonline.com/tw/word/reward)（v.））中反推出來的。  

“回報或支付服務、艱辛等的報酬”，也指“因功績、美德等而給予的獎勵”，始於14世紀晚期。有時也指“懲罰，對惡行的補償”。1590年代開始，“爲捕獲罪犯或逃犯或尋回失物而支付的一筆錢”的意義出現。  

reward 是 [regard](https://www.etymonline.com/tw/word/regard)（n.）的同源詞，在中古英語中也用於現在賦予該詞的意義：“關注、留意、注意、觀察”，也指“尊重、尊敬”。

## reward (v.)
約在1300年，源自 rewarden，表示“頒發，贈予”； 14世紀初，“贈予作爲獎品或補償”，來自盎格魯-法語和老北法語的 rewarder “注重、獎賞”（古法語 regarder “注意、關注、保護”）。這源自 re-，可能是這裏的一個強調前綴（見 [re-](https://www.etymonline.com/tw/word/re-) ），加上 warder “看、關照、看管”，來自法蘭克或其他日耳曼語言，源自原始日耳曼語中的 *wardon “保護”（源自 PIE 根詞 [*wer-](https://www.etymonline.com/tw/word/*wer-) （3）“察覺、小心注意”）。  

起初表示任何形式的回報，好的或壞的，是 [regard](https://www.etymonline.com/tw/word/regard) 的雙形式詞（動詞）。14世紀至15世紀， reward 的意義與現在賦予該詞的意義相同： “看； 關心； 考慮”。相關詞彙： Rewarded； [rewarding](https://www.etymonline.com/tw/word/rewarding) 。
