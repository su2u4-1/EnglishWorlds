# request
## request (n.)
14世紀中期， requeste，“請求幫助、服務等的行爲； 表達對某事被授予或完成的願望”，源自古法語 requeste（現代法語 requête）“請求”，來自通俗拉丁語 *requaesita，源自拉丁語 requisita (res) “被要求的（事物）”，來自 requisitus 的女性形式“被請求的，被要求的”，是 requirere “尋求知識，詢問，要求”的過去分詞（參見 [require](https://www.etymonline.com/tw/word/require)）。  

從14世紀後期開始，“被請求的東西”。到1928年，“一封信、電話等，請求在廣播節目中播放特定的歌曲，通常附有個人信息或致辭”。

## request (v.)
1530年代，“請求（某人）（做某事），表達希望某事被完成”； 1560年代，“表達願望或要求，請求被允許做某事”，源自 [request](https://www.etymonline.com/tw/word/request)（名詞）或來自法語 requester，“再次請求，要求，收回”，源自 requeste。更早的動詞是中古英語 requeren（14世紀），源自古法語 requerre，直接源自拉丁語 requiare。相關： Requested； requesting。
