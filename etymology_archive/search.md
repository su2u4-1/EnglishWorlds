# search

## search(v.)
約於1300年， serchen，“詳盡仔細地通過和檢查”（及物動詞），源自於古法語 cerchier “搜尋”（12世紀，現代法語 chercher），來自拉丁語 circare “四處遊蕩，穿行”，在晚期拉丁語中表示“在這裏和那裏遊蕩，四處走動，探索”，來自 circus “圓圈"（參見 [circus](https://www.etymonline.com/tw/word/circus)）。與西班牙語同源詞 cercar “環繞，包圍”相對應。
“仔細檢查”指對人、包等進行檢查，始見於15世紀初。短語 search me 作爲一種無奈的口頭表示於1901年記錄下來。Search engine 可以追溯到1988年。作爲描述語的短語 search-and-destroy 出現於1966年的美國英語中，是越南戰爭時期的杜撰。Search-and-rescue 出現於1944年。

## search(n.)
約在1400年， serche，指“搜尋的行爲； 尋找或觀察； 在某個區域或地方進行搜索； 檢查記錄、遺囑等； ”15世紀初，“調查非法活動的權利； ”來自盎格魯-法語的 serche，古法語的 cerche “調查”，源自 cerchier（見 [search](https://www.etymonline.com/tw/word/search)（v.））。
Search-warrant，由當局授權給警察進入嫌疑人的住所（最初特別是爲了找回被盜物品），可追溯至1739年。Search-party “尋找失物的隊伍”於1854年出現（在 Elisha Kent Kane 關於美國遠征尋找約翰·富蘭克林爵士的報告中）。
