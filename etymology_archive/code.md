# code
## code (n.)
公元1300年左右，"法律的系統化編纂"，源自古法語 code "法律體系，法律書"（13世紀），源自拉丁語 codex  "法律的系統分類"，更早的是 caudex "書"，字面意思是"樹幹"，因此指用木板塗蠟寫字的書。德·瓦恩通過原意爲 *kaud-ek- 的原意追溯到原始意大利語 *kehu-d- " "分離，分割"，他還將其視爲 cauda "尾巴"的詞根（參見 [coda](https://www.etymonline.com/tw/word/coda)）。  

意思是"密碼，信號系統及其使用規則"（secret code 中的意思）始於1808年。Code-name 始於1879年（在電報中）。"以一種計算機可用的形式表達信息和指令的系統"的意思始於1946年。  

---
## code (v.)
“將…編碼”，1815年，源自 [code](https://www.etymonline.com/tw/word/code)（n.）。具體指“將…編寫成計算機代碼”始於1947年。不及物動詞“編寫計算機代碼”始於1987年。相關詞彙： Coded； coding。
