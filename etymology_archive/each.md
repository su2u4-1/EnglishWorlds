# each

## Origin and history of each
### each

古英语中的 ælc（名词、代词、形容词）意思是“任何、所有、每一个”，它是 a-gelic 的缩写，意为“永远一样”。这个词由 a（意为“永远”，参见 [aye](https://www.etymonline.com/word/aye) (adv.)）和 gelic（意为“相似的”，参见 [like](https://www.etymonline.com/word/like) (adj.)）组合而成。它源自一个普遍的西日耳曼语表达 \*aina-galīk，这个表达也衍生出了荷兰语的 elk、古弗里斯语的 ellik、古高地德语的 iogilih 以及现代德语的 jeglich，都表示“每一个”。最初的用法与我们现在使用的 [every](https://www.etymonline.com/word/every)（这是 each 的复合词）或 [all](https://www.etymonline.com/word/all) 相似；现代用法受到拉丁语 quisque 的影响。现代拼写大约在16世纪末期才出现。此外，还可以参考 [ilk](https://www.etymonline.com/word/ilk)、[such](https://www.etymonline.com/word/such) 和 [which](https://www.etymonline.com/word/which)。
