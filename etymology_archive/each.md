# each
## each 
古英語 ælc（名詞，代詞，形容詞）“任何，所有，每個”，縮寫自 a-gelic “永遠相同”，源自 a “永遠”（見 [aye](https://www.etymonline.com/tw/word/aye)（副詞））和 gelic “相同”（見 [like](https://www.etymonline.com/tw/word/like)（形容詞））。來自一個共同的西日耳曼表達 *aina-galīk（源頭還包括荷蘭語 elk，古弗里斯蘭語 ellik，古高地德語 iogilih，德語 jeglich “每個，每個”）。最初用於我們現在使用的 [every](https://www.etymonline.com/tw/word/every)（這是 each 的複合詞）或 [all](https://www.etymonline.com/tw/word/all)； 現代用法受到拉丁語 quisque 的影響。現代拼寫出現在1500年代末。另請參見 [ilk](https://www.etymonline.com/tw/word/ilk)， [such](https://www.etymonline.com/tw/word/such)， [which](https://www.etymonline.com/tw/word/which)。
