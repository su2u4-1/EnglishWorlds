# visual

## visual(adj.)
在15世紀初，「與視覺有關」的意思開始出現；同時也有「來自眼睛或視覺的」意涵（例如光束被認為是如此），以及「由眼睛產生的，通過目光所產生的」。這些意義源自於晚期拉丁語的 visualis，意指「視覺的」，而其根源則是拉丁語的 visus，意為「視覺，注視；視力；所見之物，外觀」。這又來自於 visus，是 videre（意為「看」）的過去分詞（參見 [vision](https://www.etymonline.com/tw/word/vision)）。

「可見的，能被視覺感知的」這一意義出現於15世紀末；而「與視覺相關」的用法則可追溯至約1600年。相關詞彙還有 Visuality。名詞形式「攝影膠卷或其他視覺展示」則記錄於1944年。至於 Visual aid，則可追溯至1911年。
