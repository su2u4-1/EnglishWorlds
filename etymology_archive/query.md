# query

## query(n.)
1530年代， quaere "一個問題"，源自拉丁語 quaere "詢問，打聽"，"作爲邊注或備忘錄中用來表示問題或疑問的常用詞，因此被視爲名詞" [世紀詞典]， quaerere 的第二人稱單數祈使語氣，意爲"尋找，尋求； 努力，力求獲得； 詢問，要求，需求"； 在比喻上指"思考，學習，探索"，可能最終源自 PIE 詞根 [*kwo-](https://www.etymonline.com/tw/word/*kwo-)，是相對和疑問代詞的詞幹。拼寫在大約1600年被 inquiry 的影響下改變爲英語化。與 [quest](https://www.etymonline.com/tw/word/quest) 相比較。

> Query stands for a question asked without force, a point about which one would like to be informed : the word is used with all degrees of weakness down to the mere expression of a doubt; as, I raised a query as to the strength of the bridge. [Century Dictionary]

> Query 代表了一個沒有強制力的問題，一個想要了解的問題：這個詞的使用程度可以從非常弱到僅僅表示懷疑； 例如，我對橋的強度提出了一個 query。[世紀詞典]

---
## query(v.)
“to question, ask questions; express doubt,” 1650年代，來自 [query](https://www.etymonline.com/tw/word/query)（名詞）。不及物動詞意義出現於1680年代。相關詞彙： Queried； querying。
