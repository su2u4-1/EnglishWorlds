# core
## core (n.)
14世紀初，“任何事物的核心或內部部分”（尤其是蘋果、梨等水果），起源不明，可能來自古法語 cor， coeur “水果的核心，萵苣的心臟”，字面意思爲“心臟”，源自拉丁語 cor “心臟”，源自 PIE 詞根 [*kerd-](https://www.etymonline.com/tw/word/*kerd-) “心臟”。  

“從樹木、土壤等中央部分切割和移除”的意思始於1640年代。鑄造中“內部模具，填充預留空洞的空間”的意思始於1730年。核物理學中，“反應堆的一部分，包含核燃料並進行反應的地方”的意思始於1949年。  

---
## core (v.)
"去除（水果的）果核"，15世紀中期，來自 [core](https://www.etymonline.com/tw/word/core)（名詞）。相關詞彙： Cored； coring。
