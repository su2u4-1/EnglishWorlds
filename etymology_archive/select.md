# select
## select (adj.)
1560年代，“因特別優秀或適合而被選擇”，源自拉丁語 selectus，是 seligere 的過去分詞，“選擇，挑選，精選； 分離，挑選”，由 se- “分開”（見 [se-](https://www.etymonline.com/tw/word/se-)）和 legere “收集，選擇”（源自 PIE 根 [*leg-](https://www.etymonline.com/tw/word/*leg-)（1）“收集，聚集”）組成。  

“精心挑選”，因此“選擇，特別優秀”（自1580年代起）。相關詞彙： Selectly； selectness。名詞“被選中的人或物，精選品”記錄於約1600年。  

---
## select (v.)
"從同類事物中挑選出一個或多個，優先選擇一個或多個，" 1560年代，來自 [select](https://www.etymonline.com/tw/word/select)（形容詞）或拉丁語 selectus。相關詞彙： Selected； selecting。
