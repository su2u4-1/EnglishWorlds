# column

## column(n.)
15世紀中期，“柱子，長形的建築支撐物”，還有“頁面的垂直分割”，來自古法語 colombe（12世紀，現代法語 colonne “柱子，支柱”），源自拉丁語 columna “柱子”， columen “頂部，高峯”的同源形式，源自 PIE 詞根 [*kel-](https://www.etymonline.com/tw/word/*kel-)（2）“突出； 山丘”。

在軍事意義上，“狹窄的部隊編隊向後延伸”，從1670年代開始，與 line 相對，後者向前延伸，深度較薄。 “爲報紙撰寫的文章”（排版的 column 內容）的意義可追溯至1785年。
