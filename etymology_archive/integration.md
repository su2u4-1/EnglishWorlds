# integration
## integration (n.)
1610年代，“將整體的部分組合在一起的行爲”，源自法語 intégration，直接源自晚期拉丁語 integrationem（主格 integratio）“更新，恢復”，是拉丁語 integrare “使完整”，也是“重新開始”的過去分詞詞幹（參見 [integrate](https://www.etymonline.com/tw/word/integrate)）。反歧視意義（與 [segregation](https://www.etymonline.com/tw/word/segregation) 相對）記錄於1934年。
