# memory
## memory (n.)
13世紀末，“回憶（某人或某事）； 記憶，意識或知覺（某人或某事）”，還有“名聲，聲譽，名望”； 源自盎格魯-法語 memorie（古法語 memoire，11世紀，“思想，記憶，回憶； 紀念，記錄”），直接源自拉丁語 memoria “記憶，回憶，記憶能力”，抽象名詞，源自 memor “留心，記憶”，源自 PIE 詞根 [*(s)mer-](https://www.etymonline.com/tw/word/*(s)mer-)（1）“記得”。  

“紀念”（某人或某事）的意義來自公元1300年左右。英語中“記憶能力； 保留無意識痕跡的有意識印象或狀態，並在與過去相關的意識中回憶起這些印象或狀態”的意義在14世紀末出現。1520年代開始有“個體意識或觀察中包含的時間長度”的意思。  

> I am grown old and my memory is not as active as it used to be. When I was younger I could remember anything, whether it had happened or not; but my faculties are decaying now and soon I shall be so I cannot remember any but the things that never happened. It is sad to go to pieces like this, but we all have to do it. ["Mark Twain," "Autobiography"]  
> 我年紀大了，記憶力不如從前那麼活躍。年輕時，我能記住任何事情，無論它是否發生過； 但現在我的能力正在衰退，很快我將只能記住那些從未發生過的事情。像這樣崩潰真是令人傷心，但我們都必須經歷這個過程。【“馬克·吐溫”，“自傳”】  
1817年開始有“被記住的東西； 在思想中固定或回憶起的任何事物”的意思，儘管19世紀對這種用法的正確性存在爭議。19世紀，這個詞在更多或更少的比喻意義上被擴展到類似的物理過程。計算機的意義，“存儲信息的設備”，始於1946年。相關詞： Memories。
