# add
## add (v.)
14世紀晚期，“將某物與另一物連接或結合在一起”，源自拉丁語 addere “添加，連接，附加，放置於”，字面和比喻意義都是由 ad “到”（見 [ad-](https://www.etymonline.com/tw/word/ad-)）和 -dere 組成的，這是一個組合形式，意思是“放置，放置”，源自 dare “給予”（來自 PIE 根 [*do-](https://www.etymonline.com/tw/word/*do-) “給予”）。  

不及物動詞的意思“做算術，做加法”也來自14世紀晚期。相關： [Added](https://www.etymonline.com/tw/word/added)； adding。 add up 來自1754年； 在比喻意義上“有意義”，則是在1942年。 Adding machine “用於計算大量金額的機器”始於1822年。
