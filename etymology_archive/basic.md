# basic
## basic (adj.)
"與基礎有關的"，最初出現於1832年的化學領域，源自 [base](https://www.etymonline.com/tw/word/base)（n.）+ [-ic](https://www.etymonline.com/tw/word/-ic)。  

---
## BASIC 
計算機語言，1964年，首字母縮略詞（首字母縮寫），“Beginners' All-purpose Symbolic Instruction Code” 的縮寫； 由匈牙利裔美國計算機科學家約翰·G·凱門伊和美國計算機科學家托馬斯·E·庫茲特發明。
