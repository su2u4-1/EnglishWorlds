# description
## description (n.)
14世紀晚期， descripcioun，“描繪或描述的行爲”，源自古法語 description（12世紀）和直接源自拉丁語 descriptionem（主格 descriptio）“表現，描述，複製”，動作名詞，來自 describere “寫下，抄錄，複製，草圖”，源自 de “向下”（見 [de-](https://www.etymonline.com/tw/word/de-)） + scribere “寫”（源自 PIE 詞根 [*skribh-](https://www.etymonline.com/tw/word/*skribh-) “切割”）。通過“代表類別或個體的特質”這一概念，出現了“類型，種類”（1781）的意義。
