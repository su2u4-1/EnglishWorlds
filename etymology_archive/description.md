# description

## Origin and history of description
### description(n.)

14世纪晚期，descripcioun，意为“描绘或描述的行为”，源自古法语的description（12世纪），直接来自拉丁语的descriptionem（主格descriptio），意为“表现、描述、复制品”。这个名词是由动词describere（意为“写下、抄写、复制、素描”）的过去分词词干派生而来。它由de（意为“向下”，参见[de-](https://www.etymonline.com/word/de-)）和scribere（意为“书写”，源自原始印欧语词根[*skribh-](https://www.etymonline.com/word/*skribh-)，意为“切割”）组合而成。通过“代表某一类或个体的特征”这一概念，后来引申出了“类型、种类”的意思（1781年）。
