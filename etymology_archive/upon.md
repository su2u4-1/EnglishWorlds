# upon
## upon (adv., prep.)
12世纪早期，“在上方并接触，向上以获取或处于……之上”，来自古英语 uppon, uppan (prep.) “在……上，接触，直到，抵抗”，作为副词“在表面上或之上；悬挂状态”，来自 [up](https://www.etymonline.com/tw/word/up) (adv.) + [on](https://www.etymonline.com/tw/word/on) (prep.)。可能受斯堪的纳维亚语源的影响，如古诺尔斯语 upp a。一般表示运动到更高的地方。  

> On, Upon.These words are in many uses identical in force, butuponis by origin (up+on) and in use more distinctly expressive of motion to the object from above or from the side.Onhas the same force, but is so widely used in other ways, and so often expresses mere rest, that it is felt by careful writers to be inadequate to the uses for whichuponis preferred. [Century Dictionary, 1895]  
> On, Upon.这些词在许多用法中作用相同，但upon从起源上（up+on）和在使用中更明确地表达从上方或侧面运动到物体的动作。On具有相同的作用，但在其他方式中使用广泛，且常常仅表达静止，因此被细心的作者认为不足以用于upon被偏好的用途。[世纪词典，1895年]  
编辑们补充说， upon “在许多情况下几乎只是 on 的同义词， up 的作用几乎或完全丧失。”
