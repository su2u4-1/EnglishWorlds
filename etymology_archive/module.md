# module

## module(n.)
1580年代，「分配的量」，這個意義現在已經過時，來自法語 module（1540年代）或直接源自拉丁語 modulus，意為「小量度」，是 modus 的 diminutive 形式，意思是「量度，方式」（來自原始印歐語根 [*med-](https://www.etymonline.com/tw/word/*med-)，意指「採取適當的措施」）。

「標準量度以調節比例」的意義出現於1620年代。「可互換部件」的含義則記錄於1955年，源於「作為建築等部件尺寸基礎的長度」的概念（1936年）；而「航天器的獨立部分」的意義則始於1961年。
