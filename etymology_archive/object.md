# object

## Origin and history of object
### object(n.)

这个词最早出现在14世纪晚期，意思是“可触知的东西，能够被感官感知或呈现的事物”。它源自古法语的 object，直接借自中世纪拉丁语的 obiectum，意为“摆在眼前的东西”（可以是心灵或视觉上的），在拉丁语中是 obiectus 的名词形式，原意是“位于前方或对面的”，在古典拉丁语中还可以指“指控”或“指责”。这个词的词根 obicere 则有“呈现、反对、阻碍”的意思，由 ob（表示“在前、朝向、对抗”，参见 [ob-](https://www.etymonline.com/word/ob-)）和 iacere（意为“投掷”，源自原始印欧语词根 [*ye-](https://www.etymonline.com/word/*ye-)，意为“投掷、推动”）组合而成。

到了15世纪初，这个词又引申出“目的、目标”的意思，来自拉丁语的 obiectus，意指“呈现于视野的事物”。1580年代，它被用来表示“认知行为所指向的对象”。而在语法上，指代“句子中表示动词作用对象的成分”这一用法则出现在1729年。

短语 No object（“不被视为重要的事物”）出现于1782年，当时 object 的意思是“障碍、阻碍”，大约在1500年左右。作为形容词时，意为“能被感官感知的”，同样出现在14世纪晚期。Object-lesson（“通过实物观察进行的教学”）则起源于1831年。

## object (related entry 1)
### object(v.)

大约在公元1400年，objecten这个词出现，意思是“提出作为反对、怀疑或批评的理由；对某个提议或推理提出反对意见”。这个词源于古法语的objecter，直接来自拉丁语的obiectus，是obiectare的过去分词，意为“作为不赞成的理由引用，反对，抵制”，字面意思是“放置或投掷在前面或对抗”，是obicere的频态形式（参见[object](https://www.etymonline.com/word/object) (n.)）。相关词汇包括Objected和objecting。
