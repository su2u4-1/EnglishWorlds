# create

## create(v.)
“創造出來”，早在15世紀就來自拉丁語 creatus，它是 creare 的過去分詞，意爲“製造、生產、生育、產生、導致”，與 [Ceres](https://www.etymonline.com/tw/word/Ceres) 和 crescere “出現、出生、增加、成長”相關，源自 PIE 詞根 [*ker-](https://www.etymonline.com/tw/word/*ker-#etymonline_v_53176) (2)“增長”。德•範（De Vaan）寫道 creare 的最初含義是“促進生長”，這在舊的文本中仍然可以找到......” 相關： Created； creating。
