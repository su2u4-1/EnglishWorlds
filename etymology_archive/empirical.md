# empirical

## empirical(adj.)
1560年代，最初用於醫學領域，指“與經驗或實驗有關的”，源自拉丁語 empiricus（名詞）“一位以經驗爲指導的醫生”，來自希臘語 empeirikos “有經驗的”，源自 empeiria “經驗； 僅憑經驗或實踐而沒有知識”，尤指醫學領域，來自 empeiros “在某事上有經驗的，通過使用證明的”，源自 en（見 [en-](https://www.etymonline.com/tw/word/en-)（2））的同化形式 + peira “試驗，實驗”，源自 PIE \*per-ya-， [*per-](https://www.etymonline.com/tw/word/*per-)（3）的帶後綴形式“嘗試，冒險”。與 [-al](https://www.etymonline.com/tw/word/-al)（1）一起使用。從1757年開始，一般意義上指“僅憑經驗指導的”。相關詞： Empirically（1640年代指“通過觀察和實驗”）。
